Programming

My most recent projects include programming and coding projects. Programming interests me not only because of the practical usage, but also because of the ecstatic moment when you create an awesome project that works better than intended after all of the hard work you put into it. It also has an increasing need for engineering projects, and is an essential skill, at some basic level or another, when using sensors and control boards. Some of my projects range from a Python Selenium code that automatically logs you into your gmail to a Raspberry Pi Zero (using Raspbian) that de-authenticates the wifi of a user and tracks the packets send between the router and the computer. These projects are just some of the beginnings, and as technology and coding advances, there could be infinite potential.
